We're open

Open Source guarantees that the system is fair and no extra data is stored.

Using Digital ID

It means no fake accounts. Only those who you need.

Privacy first

We use two isolated servers to provide the highest level of protection for your personal data.

How it works

1. You authenticate on the first server using DigID and get an anonymous ticket for vote on second server.

2. You vote on second server using an anonymous ticket. As result you'll get a signed confirmation of your vote.

3. We publish every vote and you can be sure that nothing has been lost. But if does, you can confirm your vote using signed confirmation.
